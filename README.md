# Sentiment Analyze Naive Bayes
Analisis Sentimen menggunakan metode Naive Bayes dan Algoritma Genetika

## Program dan Library
- Python 3.6.8
  - Sqlite3
  - Tweepy
  - scikit-learn

## Algoritma Program
Algoritma yang digunakan pada sentimen analisis ini menggunakan metode Naive Bayes
